## Nuclear Reactors 
Problem Code: NUKES

Description:

In: A (0->1e9), N(0->100), K(1->100)

Out: K chu so (theo thu tu nguoc) cua so A (he 10) sau khi chuyen sang he N+1 (B).

Chu y:
- N = 0 -> in ra tat ca so 0
- K > so digits cua B -> fill bang 0
- in theo thu tu nguoc

Ex:
```
3 -> 11
k = 3 -> cout: 1 1 0
```

## Buying Sweets

Description:

Input:

- a[i] (1->100)

```
sum(a) >= k*X
sum(a) <= (k+1)*X
sum(a)- min(a) < k*X
```
